## Usage

```sh
gcc *.c -o server
mv server ./test
cd test && ./server
```

then `curl http://localhost:3900` or open http://localhost:3900/ in browser

index.html -> http://localhost:3900/

hello.html -> http://localhost:3900/hello.html
